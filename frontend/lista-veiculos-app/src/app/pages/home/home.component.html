Bem-vindo!

Esta página foi criada para demonstrar minhas capacidades e habilidades como desenvolvedor front-end, com o objetivo de concorrer à vaga na Info Sistemas. Aqui, você encontrará uma aplicação web moderna e responsiva, desenvolvida utilizando as melhores práticas e tecnologias atuais.

A funcionalidade principal, como solicitado, é o "CRUD" da lista de veículos. Fiquem à vontade para analisar da melhor forma.

Resumo do Back-End

Configuração Inicial

Node.js e Express foram utilizados para configurar o servidor web.

Instalação de Dependências

Mongoose foi utilizado para facilitar a interação com o MongoDB.

Estrutura de Pastas

Organização modular do código para facilitar a manutenção e escalabilidade.

Configuração do Servidor

dotenv foi utilizado para carregar variáveis de ambiente do arquivo .env.

Mongoose Connect foi utilizado para conectar ao MongoDB.

Modelo de Dados

veiculoSchema define a estrutura dos documentos na coleção de veículos.

Um campo `id` personalizado foi implementado para gerar IDs únicos e rastreáveis.

Controladores

Funções para criar, ler, atualizar e deletar veículos foram implementadas.

Rotas

Rotas para criar, ler, atualizar e deletar veículos foram definidas.

Testes Unitários

Mocha e Chai foram utilizados para garantir a funcionalidade correta do Back.

Resumo do Front-End

Configuração Inicial

Angular 17 foi utilizado como framework principal para a construção da aplicação web.

Estilos e Responsividade

SASS foi utilizado como pré-processador de CSS.

NgBootstrap foi utilizado para componentes responsivos.

Ícones

Material Icons foi utilizado para adicionar ícones modernos e atraentes à interface.

Funcionalidades

Implementação de CRUD para a lista de veículos, incluindo adicionar, editar e deletar veículos.

Serviço HTTP

Serviço para consumir a API do backend e realizar operações CRUD.

Consumo de dados

Criei uma rota para consumir as informações de um banco NoSQL (Para utilizarmos no dia da entrevista) e outra para consumir de um serviço Mockado (Para vocês testarem no projeto GitHub).

Testes Unitários

Karma e Jasmine foram utilizados para garantir a funcionalidade correta do Front.
